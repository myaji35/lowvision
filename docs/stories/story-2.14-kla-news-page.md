## 디자인 요구사항

### 레이아웃
- 상단: 카테고리 탭/필터 + 검색창
- 메인: 소식 카드 그리드 (2-3열)
- 각 카드: 이미지 (optional), 카테고리 뱃지, 제목, 날짜, 요약
- Pagination (하단)
- 상세 페이지: 본문 + 첨부파일 + 공유 버튼

### 접근성
- 시맨틱 HTML (`<article>`, `<time>`, `<nav>`)
- 카테고리 필터에 `<button role="tab">` 또는 `<nav>` 사용
- 날짜에 `<time datetime="...">` 사용
- 이미지에 명확한 대체 텍스트
- Pagination에 적절한 ARIA 라벨
- 고대비 모드에서 카테고리 뱃지 구분 명확
- 스크린 리더 친화적 마크업

### 데이터 모델
```typescript
interface NewsItem {
  id: string;
  slug: string;                     // URL용 slug (SEO)
  category: NewsCategory;
  title: string;
  summary: string;                  // 요약 (1-2줄)
  content: string;                  // 본문 (Markdown 또는 HTML)
  author?: string;                  // 작성자 (optional)
  publishedAt: Date;
  updatedAt?: Date;
  isPinned: boolean;                // 상단 고정 여부
  imageUrl?: string;                // 대표 이미지 (optional)
  imageAlt?: string;                // 이미지 대체 텍스트
  attachments?: Attachment[];       // 첨부파일
  tags?: string[];                  // 태그 (검색용)
  views?: number;                   // 조회수 (optional)
}

type NewsCategory =
  | 'announcement'    // 공지사항
  | 'event'           // 행사
  | 'program'         // 프로그램
  | 'press-release';  // 보도자료

interface Attachment {
  id: string;
  fileName: string;               // '협회_행사_안내.pdf'
  fileSize: number;               // bytes
  fileType: string;               // 'application/pdf'
  downloadUrl: string;
}

interface NewsListParams {
  category?: NewsCategory;
  search?: string;
  page?: number;
  limit?: number;                 // 페이지당 항목 수 (기본 12)
}

interface NewsListResponse {
  items: NewsItem[];
  total: number;
  page: number;
  totalPages: number;
}
```

## 주요 기능

### 1. 카테고리 필터링
사용자가 상단 탭에서 카테고리를 선택하면 해당 카테고리의 소식만 표시

**카테고리:**
- 전체 (기본)
- 공지사항
- 행사
- 프로그램
- 보도자료

---

### 2. 검색 기능
제목, 본문, 태그에서 키워드 검색

**예시 검색어:**
- "재활"
- "취업"
- "스마트폰"

---

### 3. 중요 공지 상단 고정 (Pinned)
`isPinned: true`인 게시물은 목록 최상단에 고정 표시 (별도 스타일)

---

### 4. Pagination
페이지당 12개 항목 표시, 하단에 페이지 번호

**Pagination UI:**
```
< 이전  1 2 3 4 5  다음 >
```

---

### 5. 소셜 공유
각 소식 상세 페이지에서 공유 버튼 제공

**지원 플랫폼:**
- Facebook
- Twitter (X)
- 카카오톡
- 링크 복사

---

### 6. 첨부파일 다운로드
PDF, HWP, DOCX 등 첨부파일 다운로드 지원

**파일 정보 표시:**
- 파일명: 재활프로그램_안내.pdf
- 파일 크기: 350KB
- 다운로드 버튼 (다운로드 아이콘 + "다운로드")

---
