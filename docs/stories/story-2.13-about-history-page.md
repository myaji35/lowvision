## 사용자 스토리

**As a** 한국저시력인협회를 처음 방문한 사용자 (저시력인 당사자 또는 가족)
**I want** 협회의 역사와 발전 과정을 시간순으로 명확히 이해하고
**So that** 협회의 신뢰성과 전문성을 확인하고, 어떤 활동을 해왔는지 알 수 있다

## 수용 기준 (Acceptance Criteria)

### 필수 요구사항
- [ ] 협회의 설립부터 현재까지 주요 이정표를 시간순으로 표시
- [ ] 연대별 구분 (예: 1990년대, 2000년대, 2010년대, 2020년대)
- [ ] 각 시기별 주요 성과 및 활동 내용 포함
- [ ] 타임라인 형식의 시각적 표현 (접근성 고려)
- [ ] 모바일/데스크톱 반응형 레이아웃
- [ ] WCAG 2.2 AA 접근성 준수
- [ ] 텍스트 200% 확대 시 레이아웃 유지
- [ ] 키보드 네비게이션 100% 지원

### 콘텐츠 요구사항
- [ ] 협회 설립 배경 및 초기 활동
- [ ] 주요 사업 및 프로그램 시작 시점
- [ ] 정부 인가/승인 등 공식 인정 내역
- [ ] 국내외 협력 및 네트워크 구축 내역
- [ ] 주요 성과 및 수상 내역
- [ ] 현재 진행 중인 주요 사업

## 디자인 요구사항

### 레이아웃
- 수직 타임라인 형식 (모바일: 왼쪽 정렬, 데스크톱: 중앙 라인 기준 좌우 교차)
- 각 항목마다 연도 + 제목 + 상세 설명 구조
- 시각적 구분선 및 포인트 마커 사용

### 접근성
- 시맨틱 HTML 사용 (`<time>`, `<article>`, `<section>`)
- 각 연혁 항목에 명확한 heading 구조
- 타임라인 순서를 논리적 순서로 제공
- 고대비 모드에서도 시각적 구분 명확
- 스크린 리더 친화적 마크업

## 기술 스펙

### 컴포넌트 구조
```
/webapp/src/app/about/history/
├── page.tsx              # 메인 페이지
├── components/
│   ├── TimelineItem.tsx  # 개별 연혁 항목
│   └── TimelinePeriod.tsx # 시대별 그룹
└── data/
    └── historyData.ts    # 연혁 데이터
```

### 데이터 모델
```typescript
interface HistoryItem {
  id: string;
  year: number;
  month?: number;
  title: string;
  description: string;
  category: 'founding' | 'program' | 'recognition' | 'partnership' | 'achievement';
  importance: 'high' | 'medium' | 'low';
}

interface HistoryPeriod {
  decade: string; // '1990s', '2000s', '2010s', '2020s'
  label: string;  // '1990년대', '2000년대'
  items: HistoryItem[];
}
```

## 콘텐츠 구조 (예시)

### 1990년대 - 협회 설립기
- 1990년: 사단법인 한국저시력인협회 설립
- 199X년: 저시력인 재활 프로그램 시작

### 2000년대 - 사업 확장기
- 200X년: 정부 지정 저시력 재활 전문 기관 인가
- 200X년: 전국 지부 확대

### 2010년대 - 전문화 시기
- 201X년: 보조공학 체험관 개관
- 201X년: 국제 저시력 컨퍼런스 개최

### 2020년대 - 디지털 전환기
- 2020년: COVID-19 대응 온라인 프로그램 전환
- 2024년: 웹사이트 리뉴얼 착수
- 2025년: 새로운 디지털 플랫폼 런칭

## 상호작용 시나리오

### 시나리오 1: 신규 방문자의 협회 탐색
1. 사용자가 "협회 소개" 메뉴에서 "연혁" 클릭
2. 연혁 페이지 로딩 (최신 → 과거 순으로 표시)
3. 사용자가 스크롤하며 시대별 주요 활동 확인
4. 특정 시기의 상세 설명 읽기
5. 협회의 역사와 전문성 이해

### 시나리오 2: 키보드 사용자의 탐색
1. Tab 키로 각 연혁 항목에 순차적으로 접근
2. Heading 단축키(H)로 시대별 섹션 빠르게 이동
3. Enter 키로 확장 가능한 콘텐츠 펼치기/접기

## 접근성 체크리스트

- [ ] 모든 연도 정보에 `<time>` 태그 사용
- [ ] 시대별 제목에 적절한 heading level 적용
- [ ] 타임라인 마커가 순수 장식용일 경우 `aria-hidden="true"`
- [ ] 순서가 중요한 리스트는 `<ol>` 사용
- [ ] 명암비 7:1 (AAA 수준) 달성
- [ ] 고대비 모드 지원
- [ ] 텍스트 200% 확대 테스트 통과
- [ ] 스크린 리더 테스트 (NVDA/VoiceOver)
