## 1. 현황 분석

### 핵심 타겟 세그먼트

#### 1차 회원: 저시력인 당사자
- **신규 진단자** (진단 후 6개월 이내)
  - 정서적으로 가장 취약한 시기
  - 정보 욕구 최고점
  - 협회 가입 최적 타이밍
- **기존 저시력인** (진단 후 6개월+)
  - 일상 적응 완료, 커뮤니티 필요
  - 보조공학/복지 정보 지속 수요

#### 2차 회원: 가족 및 보호자
- **부모** (아동/청소년 저시력인)
  - 교육/진로 정보 필요
  - 심리적 지원 및 부모 커뮤니티
- **성인 자녀** (노인 저시력인)
  - 부모 케어 방법
  - 복지 혜택 신청 지원
- **배우자**
  - 일상생활 지원 방법
  - 부부 관계 유지 전략

#### 3차 회원: 전문가 및 지원자
- 재활 전문가, 사회복지사
- 특수교사, 보조공학 전문가
- 안과 의료진 (협력 네트워크)

### 현재 도전과제

| 영역 | 도전과제 | 영향 |
|------|---------|------|
| **인지도** | 신규 진단자의 협회 존재 자체 인지 부족 | 진단 후 6개월 "골든타임" 놓침 |
| **접근성** | 디지털 접근성 장벽 (특히 50대+ 노인) | 온라인 가입 중도 포기율 높음 |
| **참여도** | 기존 회원의 낮은 활동 참여율 | 휴면 회원 비율 70%+ 추정 |
| **정보 격차** | 수도권 vs 지방, 청년 vs 노인 정보 불균형 | 특정 세그먼트만 혜택 집중 |
| **리소스** | 협회 인력 부족으로 회원 관리 미흡 | 1:1 관계 형성 어려움 |

## 2. 목표 설정

### 1년 로드맵 (2025년 기준)

| 지표 | 현재 | 3개월 후 | 6개월 후 | 12개월 후 |
|------|------|---------|---------|-----------|
| **신규 회원** | 기준선 | +100명 | +200명 | +500명 |
| **월간 활성 회원 (MAU)** | - | 20% | 30% | 50% |
| **온라인 커뮤니티 게시물** | 주 0-1개 | 주 3개 | 주 5개 | 주 15개 |
| **오프라인 행사 참석** | 회당 10명 | 회당 20명 | 회당 30명 | 회당 50명 |
| **이메일 뉴스레터 오픈율** | - | 20% | 25% | 35% |
| **보조공학 체험 프로그램 참여** | 0명 | 분기 10명 | 분기 25명 | 분기 50명 |

### 성공 정의

**정량적 지표**:
- 12개월 내 회원 500명 증가
- 월간 활성 회원(MAU) 50% 달성
- 회원 추천을 통한 신규 가입 30% 이상

**정성적 지표**:
- 진단 직후 "협회를 먼저 찾는" 문화 형성
- 회원 간 자발적 멘토링 네트워크 구축
- "저시력인의 정보 허브"로 브랜드 인식 확립

## 3. 회원 증대 전략

### A. 진단 시점 개입 (병원 연계)

**목표**: 신규 진단자를 진단 후 1주일 내 접촉

#### 실행 방안

**1. 안과 병원 협력 프로그램**
```
대상 병원:
- 대학병원: 서울대병원, 삼성서울병원, 세브란스, 서울아산병원
- 전문 안과: 누네안과, 강남밝은안과, BGN밝은눈안과 등

협력 모델:
① 진료실 내 "진단 직후 케어 패키지" 비치
   - 협회 소개 브로셔 (대활자 인쇄 + 점자)
   - QR코드 → /diagnosis/first-steps 직접 연결
   - "첫 전화 상담 무료" 쿠폰

② 병원 안내 데스크 협회 포스터 부착
   - "저시력 진단받으셨나요? 혼자가 아닙니다"
   - 협회 연락처 대활자 표기

③ 의사-환자 상담 시 구두 안내 요청
   - "한국저시력인협회라는 곳이 있습니다" 한 문장 추가
   - 진료비 고지서에 협회 연락처 인쇄
```

**측정**: 병원 경로 신규 회원 월 20명 이상

**2. 공공 재활병원 연계**
```
대상:
- 서울/경기/부산 어린이재활병원
- 국립재활원 저시력센터

프로그램:
- 재활 훈련 등록 시 협회 회원 가입 안내 (패키지 제공)
- 월 1회 협회 전문가 "보조공학 체험 세션" 공동 진행
- 재활 수료자 대상 "졸업 선물" (협회 굿즈 + 6개월 무료 회원)
```

**3. 디지털 검색 광고**
```
플랫폼:
- 네이버 검색 광고 (파워링크)
- 구글 애즈 (키워드 광고)
- 카카오 모먼트

키워드 전략:
[긴급성 키워드 - 입찰가 高]
- "황반변성 진단", "시력 갑자기 나빠짐"
- "녹내장 실명", "망막색소변성증 초기"

[정보 탐색 키워드 - 입찰가 中]
- "저시력 복지 혜택", "시각장애 보조기기"
- "저시력인 재활", "시각장애 일상생활"

랜딩 페이지:
- /diagnosis (진단 단계) → 하단에 "무료 상담 신청" CTA
- /rights/welfare-benefits (복지 단계) → "회원 가입하고 1:1 컨설팅 받기"

예산: 월 50만원 (연 600만원)
목표: 월 30명 신규 가입 (CPA: 16,667원)
```

### B. 온라인 커뮤니티 확산

**목표**: 저시력인 당사자의 자발적 추천 문화 형성

#### 실행 방안

**1. 카카오톡 오픈채팅 채널 세분화**
```
채널 구성 (총 6개):

[질환별]
1. "황반변성 당사자 모임" (50대+ 중심)
   - 매주 수요일 오후 3시 "질문의 시간"

2. "녹내장 당사자 네트워크" (40-60대)
   - 안압 관리 팁 공유

[세대별]
3. "저시력 청년 네트워크" (20-40대)
   - 취업/연애/여행 정보
   - 비공개 채팅방 (심리적 안전 보장)

4. "저시력 자녀 부모 모임"
   - 교육/진로 정보
   - 양육 스트레스 공감대

[주제별]
5. "보조공학 정보 공유방"
   - 신제품 리뷰
   - 중고 거래 (협회 인증)

6. "저시력 여행/취미 모임"
   - 접근성 좋은 여행지 정보
   - 함께하는 문화생활

운영 방식:
- 각 채팅방에 협회 스태프 1명 상주 (모더레이터)
- 월 1회 "이달의 협회 혜택" 공지
- 분기 1회 오프라인 모임 (선택 참여)
```

**2. 네이버 카페/커뮤니티 연계**
```
대상 카페:
- "황반변성카페" (회원 5,000명+)
- "녹내장 환우회" (회원 3,000명+)
- "시각장애인 정보나눔"

협력 방식:
- 협회 전문가가 월 2회 "전문가 Q&A" 세션 진행
- 카페 회원 대상 "협회 회원 특별 할인" (보조공학 체험 등)
- 우수 답변자 "KLA 명예 홍보대사" 임명 (소정의 활동비)

콘텐츠 전략:
- "이번 주 핫 이슈" 요약 포스팅 (복지 정책 변경 등)
- "보조공학 신제품 리뷰" (유튜브 연계)
```

**3. 유튜브 채널 운영**
```
채널명: "저시력인TV - 한국저시력인협회"

콘텐츠 포맷:

[시리즈 1] "저시력 브이로그" (주 1회)
- 저시력인 당사자의 일상 (출퇴근, 요리, 여행)
- 당사자 직접 촬영 (스마트폰 접근성 기능 활용)
- 목표: 구독자 1만명 (12개월)

[시리즈 2] "보조공학 언박싱" (월 2회)
- 최신 보조기기 리뷰 (Envision Glasses, OrCam 등)
- 실제 사용자 피드백

[시리즈 3] "전문가 인터뷰" (월 1회)
- 안과 의사, 재활 전문가, 복지 담당자
- "궁금했던 것 모두 물어보기"

수익화:
- 구독자 1,000명 달성 시 유튜브 광고 수익 → 협회 운영비
- 협찬 제품 리뷰 (보조공학 기업과 파트너십)
```

**4. 추천 리워드 프로그램**
```
"친구와 함께 성장하기" 프로그램

혜택:
[추천인 (기존 회원)]
- 1명 추천 시: 협회 소식지 1년 무료 (우편 배송)
- 3명 추천 시: 보조공학 체험권 (1회, 2주)
- 5명 추천 시: 연례 행사 VIP 초대 (동반 1인)

[신규 회원 (피추천인)]
- 가입 즉시: "저시력 생활 가이드북" PDF 다운로드
- 첫 달: 무료 전화 상담 (30분, 전문 상담사)
- 3개월 후: "3개월 생존 키트" (돋보기, 독서대 등 소품)

추천 방법:
- 웹사이트 "내 추천 링크" 생성 (/members/referral)
- 카카오톡 공유 버튼 클릭
- 오프라인: 추천 카드 (명함 크기) 배포

목표: 신규 회원의 30%를 추천으로 확보
```

### C. 오프라인 접점 확대

**목표**: 지역별 오프라인 거점 마련, 디지털 소외 계층 포용

#### 실행 방안

**1. 지역별 "저시력 정보의 날" (분기별)**
```
개최 지역: 서울 → 경기 → 부산 → 대구 순회

프로그램 (10:00-16:00):

[오전 - 정보 부스]
- 복지 상담 부스 (장애인 등록, 활동지원)
- 보조공학 체험존
  • AI 안경 (Envision Glasses)
  • 스마트 지팡이 (WeWALK)
  • 비디오 확대기
  • 스마트폰 접근성 설정 실습

[오후 - 세미나]
- 14:00 "올해의 새 복지 정책" (사회복지사)
- 15:00 "당사자 경험담" (패널 3명)
  • 취업 성공 사례
  • 독립 생활 사례
  • 보조공학 활용 사례

참여 혜택:
- 현장 가입 시: 협회 에코백 + 대활자 수첩
- 사전 등록 시: 점심 도시락 제공
- 추첨: 보조기기 무료 체험권 (10명)

홍보:
- 지역 복지관 게시판
- 안과 병원 포스터
- 네이버 카페 공지 (3주 전부터)

목표: 회당 50명 참석, 현장 가입 20명
```

**2. 공공기관 협력 네트워크**
```
[주민센터 복지 상담 연계]
- 전국 주요 주민센터에 "협회 소개 리플렛" 비치
- 복지 담당자 대상 "저시력 이해 교육" (온라인 30분)
- 복지 상담 시 협회 연락처 안내 요청

[장애인복지관 정기 세미나]
- 분기 1회 "저시력인 권익 세미나" 공동 주최
- 복지관 회원 대상 협회 가입 우대 (회비 할인)

[시/도 장애인권익옹호기관]
- 저시력 관련 권익 침해 사례 공유
- 협회 법률 자문단과 협력 체계
```

**3. 대학병원 "저시력 클리닉의 날"**

## 4. 회원 활성화 전략

### A. 회원 여정별 맞춤 콘텐츠 (Lifecycle Marketing)

```
회원 여정 5단계:

┌─────────────────────────────────────────────────────┐
│ 1단계: 신규 가입 (D+0)                               │
│ ─────────────────────────────────────────────────── │
│ - 환영 이메일 자동 발송                              │
│ - "회원님을 위한 첫 3단계" 안내                      │
│   ① 복지 혜택 확인 (/rights/welfare-benefits)      │
│   ② 내 지역 지원 프로그램 찾기                       │
│   ③ 카카오톡 채팅방 참여                            │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
```
